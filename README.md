🚀 Space Explorer - Unity Project
A 3D space flight simulator developed in Unity. Navigate through a solar system while avoiding fatal collisions with celestial bodies using cinematic movement logic.

🕹️ Flight Controls:
Action	Key	Aeronautical Axis
Thrust (Forward)	Space	Forward Movement
Pitch W/S or	Arrows Up/Down 
Yaw	 A/D or Arrows Left/Right
Roll	Q / E	Rotation
Shoot	Left Click (Mouse 0)	Projectile Launch
